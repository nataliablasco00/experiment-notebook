Using image compression plugins¶

If you design compression experiments, you may want to compare you methods with others publicly available.

The plugins folder of the library contains enb-compatible implementations of image compressors (typically wrappers for binary compressors/decompressors).

Note

For some of the codecs, only the enb wrapper is provided but not the compressor binaries. This is due to IPR restrictions.

In these tables, the lossless range refers to any data type.
